# Tetris Game

This game has 2 modes:
 - Full mode - via GUI
 - Basic mode - via CLI

The GUI mode uses the `pygame` library, which is listed in `requirements.txt`.

## Running the game
First you'll need to: `cd src/`

Then, for the GUI version, run:
`python main.py`

For the CLI version, run:
`python main.py`
